# Food Roulette｜今日食咩好 v3

喵主廚老虎機新版：首頁放大老虎機，改用拉桿啟動抽籤，抽籤次序為：飲品 → 食品 → 餐廳。

## 更新內容

- 首頁重製為大型老虎機介面
- 新增右側拉桿按鈕
- 抽籤流程改為：先抽飲品，再抽食品，最後抽餐廳
- 加入老虎機震動動畫、燈泡閃爍、拉桿動畫
- 手機支援時可啟用震動特效
- 保留餐廳為主的資料庫
- 每間餐廳可批量輸入食品及飲品
- 預設大型連鎖餐廳資料

## 安裝

```bash
npm install
npm run dev
```

## Vercel

Output Directory: `dist`
